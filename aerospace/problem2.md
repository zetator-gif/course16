## Rankine temperatuurschaal
bij absolute temperatuurschalen is het absoluut nulpunt gelijkgesteld ann 0  <br>
**hieruit** volgt dat **0°R** = **0°K**  <br>
**tevens** is een temperatuurverschil van **1°R** gelijkgesteld aan een temperatuurverschil van **1°F**  <br>

## Degree Symbol `°`
hold **`Alt`** and type **`248`** → **`°`** on Windows

## Greek letter `ρ`(rho)
alt code not working  <br>
github render using **HTML-style entities**:  <br>
• **Decimal code**: `&#961;` → &#961;  <br>
• **Hex code**: `&#x03C1`  → &#961;  <br>
## Greek letter `γ`(gamma)
• **Decimal code**: `&#947;` → &#947;  <br>
## Greek letter `θ`(theta)
• **Decimal code**: `&#952;` → &#952;  <br>
